# 親指シフト用kanata構成

[kanata](https://github.com/jtroo/kanata)で利用できる親指シフト用構成です．以下のファイルのいずれかをkanataで読み込んで下さい．かな入力で利用したい場合は1行目をコメントアウトして2行目をアンコメントしてください．

- `oyayubi_single.kbd`: 親指キーの長押しを許容しない．
- `oyayubi.kbd`: 親指キーの長押しを許容する．オススメ．
- `oyayubi_single_without_bspc`: `oyayubi_single.kbd`において余分な出力とバックスペースを伴わないバージョン．タイピングソフトなどでも利用できる．
- `oyayubi_without_bspc`: `oyayubi.kbd`において余分な出力とバックスペースを伴わないバージョン．タイピングソフトなどでも利用できる．


## 注意事項

- 半角全角キー以外でIMEオンオフを操作するなどしてこの設定の想定とずれてしまった場合，親指シフトキーを押しながら半角全角キーを押すことで元の半角全角キーを押すことと同じ挙動をとらせることができ，ずれを直すことができます．
- 特別なショートカットキーとして`ctl + スペース`，`ctl + 変換`に親指シフトモードオンオフ(IMEオンオフ)を割り当てています．
- fcitx5などでアプリケーション毎に入力メソッドを切り替えられる設定の場合は意図した挙動になりません，入力メソッドをグローバルで共有してください．
- windowsなどIMEオンオフを同期できない場合は， 意図した挙動をとるためにはアクティブウィンドウのIMEオンオフを監視してkanataのTCPサーバーに通知するクライアントを作成する必要があります．
- windowsでは旧IMEの方が安定的に動作します．
- windowsではkanataでIntlRo,IntlYenが利用できないため，かな入力で出力できない文字があります．ローマ字入力を用いてください．
